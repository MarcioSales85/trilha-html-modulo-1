História dos computadores

Os primórdios da tecnologia

(Início)

Nos primórdios das tecnologias relacionadas ao surgimento dos computadores vemos que os primeiros computadores que surgiram tiveram propósitos bem específicos para sua utilização, desde atender a redes locais como a utilização como ferramentas de informação que auxiliaram os militares em períodos de guerra. Os primeiros computadores demandavam espaços gigantescos. Alguns precursores como: 1Grace Hopper* e Joseph Licklider, que estiveram presentes na utilização do primeiro computador, Mark I, fizeram grandes contribuiçoes importantes para a área de tecnologia, esses predederam nomes conhecidos pela geração atual como Bill Gates e Steve Jobs.

1 Grace Hopper, aplicou o termo 'bug' para a área de tecnologia.

Relação entre Clients e Servs.

(Início)

Clients.

Podemos considerar como os clientes que consomem os serviçoes oferecidos pela web. Esses, podem solicitar acesso a sites através de várias plataformas como notebooks, tablets, smartphones e desktops.

Devido existirem tantas plataformas de acesso a web, é importante para o programador apresentar soluções responsivas na hora de criar os websites. Os especialistas que atuam no lado dos clients, também são conhecidos como Front End.

Servs.

Os sites quando são desenvolvidos precisam está disponíveis para os usuários, dessa forma os servidores tem a função de armazenar os sites que serão disponibilizados para os clientes. Existem servidores como várias finalidades, como exemplo:

  • Arquivos
  • Segurança
  • Streaming
  • E-mail
  • Web

Existe um trâmite de informações que ocorrem entre os servidores e o banco de dados para atender as demandas que vem dos navegadores, nesse processo o Back End trabalha aplicando as regras e garantindo um ambiente seguro aos usuários.
